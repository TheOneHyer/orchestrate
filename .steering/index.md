# Orchestrate - Steering Index

**Orchestrate** is an enterprise-grade web application for managing corporate training programs. It features multi-view scheduling, trainer burnout monitoring, wellness/recovery tracking, course catalogs, and analytics.

This `.steering` directory contains AI-assistant-oriented analysis and strategy documents to help navigate and contribute to this repository effectively.
